DOM Manipulation

What is the DOM? DOM stads for Document Object Model. It is a way of manapulating object of an html document. It is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document.

Adding element to the DOM

The are two different ways of adding an element to the page. The two ways are the append and appendchild method, we select the element we want to add to the DOM and append an element to it. In appending there is append and appendchild.

Different between append and appendc child

The different between append and appendchild is that in append, we can append divs, spans, strig or even mltiple strings; while appendchild, we can only append divs and spans but not strings.
